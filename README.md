# node-redis

Firstly please install

```
redis
```

After that execute

```
redis-server
```

Then you can clone this repository
```
git clone
```

Secondly execute
```
npm install or yarn
```

After that execute
```
node .
```

or if you already install nodemon then execute
```
npm start or yarn start
```

Then you could use postman or use browser, the example as picture below
